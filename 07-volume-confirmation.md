# 第七章：成交量验证K线形态

## 7.1 量价关系基本原理

**核心原则**：成交量为价格行为提供"确认"或"背离"的证据

### 量价配合（确认）
- 上涨 + 放量 = 健康上涨，趋势可持续
- 下跌 + 放量 = 恐慌性下跌，可能加速赶底
- 突破 + 放量 = 有效突破，突破可信

### 量价背离（警告）
- 上涨 + 缩量 = 上涨动力减弱，可能见顶
- 下跌 + 缩量 = 下跌动力衰竭，可能见底
- 创新高 + 成交量下降 = 顶背离，反转信号
- 创新低 + 成交量萎缩 = 底背离，反转信号

## 7.2 不同形态的成交量特征

### 底部反转形态的成交量
| 形态 | 典型成交量特征 |
|------|--------------|
| 锤子线 | 盘中放量下跌后缩量回升，或下影线部分放量 |
| 看涨吞噬 | 第二根阳线成交量 > 第一根阴线 |
| 启明星 | 第三根阳线成交量明显放大 |
| 刺透形态 | 第二根阳线成交量放大 |
| 双底/W底 | 右底成交量 < 左底（抛压减轻），突破颈线放量 |
| 头肩底 | 右肩成交量最小，突破颈线巨量 |

### 顶部反转形态的成交量
| 形态 | 典型成交量特征 |
|------|--------------|
| 上吊线 | 盘中放量下跌，拉回时缩量 |
| 看跌吞噬 | 第二根阴线成交量 > 第一根阳线 |
| 黄昏星 | 第三根阴线放量（有时也出现在第一根） |
| 乌云盖顶 | 第二根阴线成交量放大 |
| 双顶/M头 | 右顶成交量 < 左顶（"价量背离"） |

### 持续形态的成交量
| 形态 | 典型成交量特征 |
|------|--------------|
| 上升三法 | 中间三根缩量，第五根放量 |
| 下降三法 | 中间缩量反弹，第五根放量下跌 |
| 旗形 | 旗面整理缩量，突破放量 |
| 三角形 | 收敛过程缩量，突破方向放量 |

## 7.3 常用量价指标

### OBV（On-Balance Volume，能量潮）
- 上涨日：累计成交量 = 前值 + 当日成交量
- 下跌日：累计成交量 = 前值 - 当日成交量
- OBV与价格同步上升 → 趋势健康
- OBV与价格背离 → 趋势可能反转

### 成交量的量度方法
- `成交量均线`：将成交量与近期均值比较（如5日均量、20日均量）
- `量比 = 当日成交量 / 近N日均量`：量比 > 1.5 为放量，> 2 为巨量
- `缩量`：成交量 < 50% 的近期均量

## 7.4 关键量价配合场景

### 场景1：底部放量反转
- 长期下跌后出现一根大阳线
- 成交量明显放大（> 前几日2倍以上）
- 大概率是主力资金入场，底部信号强

### 场景2：缩量回调后的放量启动
- 放量突破关键阻力位（年线、颈线等）
- 随后缩量回踩不破突破位
- 再次放量启动 → 经典"三浪"上涨启动

### 场景3：高位放量滞涨
- 价格在高位徘徊，成交量持续放大但价格不创新高
- 这是典型的"出货"信号，风险极高

### 场景4：缩量反弹（"量价背离"的最危险形态）
- 价格反弹但成交量持续萎缩
- 反弹不可持续，随时会再次下跌

## 7.5 缠师论成交量

缠中说禅原文核心观点：
- "放量突破年线然后缩量回调年线的老股，这都是以后的黑马"
- 放量突破是"表态"，缩量回调是"确认"，放量启动是"进攻"
- 成交量是走势的"辅助验证"，价格结构才是第一位的
