# 第四章：三K线形态

## 4.1 启明星（Morning Star）

**形态特征**：
1. 出现在下跌趋势底部
2. 第一根K线：长阴线（延续下跌）
3. 第二根K线：小实体K线（十字星、纺锤线等），与第一根可能有跳空缺口
4. 第三根K线：长阳线，收盘价至少深入到第一根阴线实体的一半以上

**含义**：
- 经典的底部反转形态，可靠性极高
- 从"黑暗"（阴线）到"黎明"（星）到"日出"（阳线）
- 三根K线如同一个"V"形反转的缩影
- 如果第二根是十字星 → 称为"早晨十字星"，信号更强
- 如果中间有跳空缺口 → 信号更强

**市场心理**：第一天空头强势；第二天多空平衡，下跌动能衰竭；第三天多头完全接管。

## 4.2 黄昏星（Evening Star）

**形态特征**：
1. 出现在上涨趋势顶部
2. 第一根K线：长阳线
3. 第二根K线：小实体K线，与第一根可能有跳空缺口
4. 第三根K线：长阴线，收盘价深入到第一根阳线实体的一半以下

**含义**：
- 经典的顶部反转形态
- 从"白天"（阳线）到"黄昏"（星）到"黑夜"（阴线）
- 如果第三根阴线收盘低于第一根阳线的开盘价 → "收盘价低于开盘价"版本，反转更可靠
- 是启明星的镜像

## 4.3 三白兵（Three White Soldiers）

**形态特征**：
1. 连续三根大阳线
2. 每根阳线的收盘价依次抬高
3. 每根阳线的开盘价位于前一根阳线的实体内部
4. 上影线短或无

**含义**：
- 强烈的看涨持续形态
- 显示多头稳步推进，趋势强劲
- 如果第三根阳线实体明显缩短或出现长上影线 → 可能已是强弩之末
- 出现在长期下跌后 → 底部反转信号
- 出现在上涨中途 → 趋势持续信号

## 4.4 三黑鸦（Three Black Crows）

**形态特征**：
1. 连续三根大阴线
2. 每根阴线的收盘价依次降低
3. 每根阴线的开盘价位于前一根阴线实体内部
4. 下影线短或无

**含义**：
- 强烈的看跌持续形态
- 空头稳步推进
- 如果出现在长期上涨后的高位 → 顶部反转信号极强

## 4.5 上升三法（Rising Three Methods）

**形态特征**：
1. 出现在上涨趋势中（持续形态）
2. 第一根K线：长阳线
3. 中间三根小阴线/小阳线，实体在第一根阳线范围内，依次小幅下跌
4. 第五根K线：长阳线，收盘价创新高

**含义**：
- 看涨持续形态
- 中间的小K线是"休整"，多头在积蓄力量
- 第五根阳线确认趋势延续
- 成交量缩量（中间）→ 放量（突破），是最佳配合

## 4.6 下降三法（Falling Three Methods）

**形态特征**：
1. 出现在下跌趋势中
2. 第一根K线：长阴线
3. 中间三根小K线在第一根阴线范围内小幅反弹
4. 第五根K线：长阴线，收盘价创新低

**含义**：
- 看跌持续形态
- 中间小阳线是"诱多反弹"

## 4.7 两阳夹一阴（多方炮）

**形态特征**：
- 阳线 → 阴线 → 阳线
- 两侧阳线实体较长
- 中间阴线实体较小且被两侧阳线包裹

**含义**：
- 看涨持续/启动形态
- "两阳夹一阴，看涨到天明"

## 4.8 两阴夹一阳（空方炮）

- 阴线 → 阳线 → 阴线
- 看跌持续形态
- "两阴夹一阳，双手往外光"

## 4.9 三K线形态强弱排序

**看涨（由强到弱）**：
1. 启明星（含早晨十字星）
2. 三白兵（底部时）
3. 上升三法
4. 多方炮

**看跌（由强到弱）**：
1. 黄昏星（含傍晚十字星）
2. 三黑鸦
3. 下降三法
4. 空方炮
