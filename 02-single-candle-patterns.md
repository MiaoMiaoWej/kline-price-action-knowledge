# 第二章：单根K线形态

## 2.1 大阳线 / 光头光脚阳线（Marubozu / Long Bullish Candle）

**形态特征**：
- 无上影线、无下影线（或极短）
- 实体很长，开盘即最低，收盘即最高
- 量化：实体长度 > ATR × 1.5

**含义**：
- 买方完全主导，从开盘持续拉升到收盘
- 强烈的看涨信号，市场情绪极度乐观
- 出现在上涨趋势中 → 趋势延续
- 出现在底部区域 → 反转确立

**注意**：如果是大阴线（光头光脚阴线），则含义完全相反，卖方完全主导。

## 2.2 锤子线（Hammer）

**形态特征**：
- 出现在下跌趋势的底部
- 小实体（阳或阴均可）
- 下影线极长（至少实体长度的2-3倍）
- 上影线极短或没有

**含义**：
- 价格在盘中大幅下探，但被强力买盘推回
- 显示卖盘衰竭，买盘开始接管
- 下影线越长，反转信号越强
- 需要后续K线验证（阳线收盘确认更佳）
- 红色（阴线）锤子线比绿色（阳线）锤子线弱

**市场心理**：空头将价格打到低位，但多头强力反击，最终价格回到高位，空头力量已经被消耗。

## 2.3 上吊线（Hanging Man）

**形态特征**：
- 出现在上涨趋势的顶部
- 形态与锤子线相同：小实体 + 长下影线 + 短上影线
- 区别仅在于 **位置**：出现在高位

**含义**：
- 看跌反转信号
- 盘中曾大幅下跌，虽然收盘拉回，但表明卖压已经出现
- 需要下一根K线收跌来确认
- 如果下一根K线跳空低开，信号更强

## 2.4 倒锤子线（Inverted Hammer）

**形态特征**：
- 出现在下跌趋势底部
- 小实体
- 上影线极长（实体长度的2-3倍）
- 下影线极短或没有

**含义**：
- 看涨反转信号（弱于锤子线）
- 盘中曾大幅上冲，虽然收盘回落，但试探了上方阻力
- 需次日阳线高开确认

## 2.5 射击之星（Shooting Star）

**形态特征**：
- 出现在上涨趋势顶部
- 小实体
- 上影线极长
- 下影线极短或没有

**含义**：
- 强烈的看跌反转信号
- 价格上涨后遭遇强大卖压，"流星坠落"
- 上影线越长，信号越可靠

## 2.6 十字星（Doji）

**形态特征**：
- 开盘价 ≈ 收盘价（实体极小或没有）
- 是市场犹豫不决的标志

### 十字星的三种主要变种：

#### (1) 蜻蜓十字星（Dragonfly Doji）
- 无上影线，长下影线
- 出现在底部 → 看涨（市场拒绝更低价格）
- 出现在顶部 → 需要警惕

#### (2) 墓碑十字星（Gravestone Doji）
- 无下影线，长上影线
- 出现在顶部 → 看跌（市场拒绝更高价格）
- 强烈的顶部反转信号

#### (3) 长腿十字星（Long-legged Doji）
- 上下影线都很长
- 多空力量极度平衡，波动剧烈
- 市场缺乏方向，往往预示重大变盘

**核心原则**：十字星出现在趋势末端（高位或低位）才有反转意义；出现在盘整区意义不大。

## 2.7 纺锤线（Spinning Top）

**形态特征**：
- 实体短小（阴阳均可）
- 上下影线长度近等
- 实体位于K线中部

**含义**：
- 市场犹豫不决，多空胶着
- 在上涨趋势末端 → 可能反转
- 在下跌趋势末端 → 可能反转
- 在盘整中 → 延续整理

## 2.8 无实体/小实体线

包括各类实体极短的K线，核心含义是"犹豫"。

| 大阴线 | 任何位置 | 看跌 | 高（趋势中） |
| 锤子线 | 下跌底部 | 看涨反转 | 高 |
| 上吊线 | 上涨顶部 | 看跌反转 | 高（需确认） |
| 射击之星 | 上涨顶部 | 看跌反转 | 高 |

| 墓碑十字星 | 顶部 | 看跌 | 高 |
