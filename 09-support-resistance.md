# 第九章：支撑阻力位置的K线形态

## 9.1 关键位置的识别

### 支撑位（Support）
- 前期低点（波谷）
- 重要均线（如20日、60日、120日、250日）
- 趋势线（上升趋势的下轨）
- 黄金分割位（如0.382、0.5、0.618）
- 成交密集区（历史上大量换手的价位区域）
- 整数关口

### 阻力位（Resistance）
- 前期高点（波峰）
- 重要均线
- 趋势线（下降趋势的上轨）
- 黄金分割位
- 成交密集区（套牢盘区域）
- 整数关口

## 9.2 形态在支撑阻力位置的放大效应

**核心规律**：支撑/阻力位附近的K线形态，信号强度是普通位置的数倍

### 为什么支撑/阻力位置信号更强？
1. **心理因素**：关键位置是市场参与者共同关注的焦点，"记忆"效应
2. **止损簇集**：关键位置附近大量止损单集中
3. **主力行为**：主力机构利用关键位置"试盘"或"洗盘"
4. **自我实现**：很多人相信这些位置有效，反而使其真的有效

### 具体示例

#### 年线附近的信号强度
- 年线处出现锤子线 → 底部反转可靠性远高于普通位置
- 年线处出现看涨吞噬 → 极有可能是大级别底部
- 缠师原文："放量突破年线然后缩量回调年线的老股，这都是以后的黑马"

#### 前高/前低附近的信号强度
- 突破前高时出现大阳线（无影线）→ 真突破概率大
- 在前高处出现射击之星 → 很可能是假突破，双顶形成
- 在前低处出现锤子线 → 双底支撑确认

#### 均线系统附近的信号
- 在60日均线处出现看涨吞噬 → 中期趋势支撑确认
- 在20日均线处出现启明星 → 短期回调结束

## 9.3 真假突破的判断

### 真突破的特征
1. **放量突破**：突破时成交量显著放大（> 前几日均量的1.5倍以上）
2. **收盘确认**：收盘价有效站在突破位之外（最好超过3%或3日站稳）
3. **K线形态配合**：突破K线为大阳线/光头光脚阳线，或看涨吞噬
4. **回踩不破**：突破后回踩不破支撑/阻力位

### 假突破（"多头陷阱/空头陷阱"）的特征
1. **缩量突破**：成交量没有放大，"虚破"
2. **长上影线**：突破瞬间被打回，留下长上影线
3. **乌云盖顶/射击之星**：在阻力位出现看跌K线形态
4. **次日反转**：突破后次日立即跌回

## 9.4 经典实战场景

### 场景1：支撑位出现看涨吞噬
- 股价下跌到关键支撑位（如前期低点、年线）
- 出现标准的看涨吞噬形态
- 成交量配合放大
- → 是极佳的买入信号

### 场景2：阻力位出现黄昏星
- 股价上涨到关键阻力位（如前期高点、前高颈线）
- 出现黄昏星形态
- → 可能形成双顶或M头，卖出信号

### 场景3：突破后的回踩确认
- 放量突破关键阻力位
- 缩量回踩不破（在突破位上方出现锤子线或刺透形态）
- 再次放量启动 → 经典"突破-回踩-启动"三浪

## 9.5 缠论视角下的支撑阻力

缠师对"历史密集区"的论述：
> "历史密集区就是历史上大多数的人都套在这个地方，股票又不是慈善晚会，那些没信心，没耐心的不下来，换手不充分，怎么可能大涨呢？"

关键点：
1. 密集成交区 = 换手充分区 = 真实支撑/阻力
2. 距离当前时间越近的密集区，效力越强
3. 时间久的密集区会被后续换手"化解"
4. 大级别的支撑阻力 > 小级别
