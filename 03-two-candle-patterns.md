# 第三章：双K线形态

## 3.1 看涨吞噬（Bullish Engulfing）

**形态特征**：
1. 出现在下跌趋势中
2. 第一根K线：小阴线（实体较小）
3. 第二根K线：大阳线，其实体完全"吞噬"第一根阴线的实体（开盘价低于前收盘，收盘价高于前开盘）

**含义**：
- 强烈的看涨反转信号
- 第二根阳线越长，吞噬越彻底，信号越强
- 如果同时吞噬前影线，称为"吞噬全部"，信号更强
- 成交量放大配合最佳

**市场心理**：第一天空头仍占主导但力量减弱；第二天开盘后空头继续打压，但多头强力反击，不仅收复失地还创出新高，空头完全被"吞噬"。

## 3.2 看跌吞噬（Bearish Engulfing）

**形态特征**：
1. 出现在上涨趋势中
2. 第一根K线：小阳线
3. 第二根K线：大阴线，其实体完全吞噬第一根阳线实体

**含义**：
- 强烈的看跌反转信号
- 第二根阴线越长，跌幅越大，信号越强
- 出现在长期上涨后或阻力位效果最佳

## 3.3 乌云盖顶（Dark Cloud Cover）

**形态特征**：
1. 出现在上涨趋势顶部
2. 第一根K线：长阳线
3. 第二根K线：长阴线
   - 开盘价高于前一天最高价（跳空高开）
   - 收盘价深入第一根阳线实体的下半部分（至少1/2以下）

**含义**：
- 看跌反转信号
- 阴线收盘深入阳线实体的程度越深，信号越强
- 如果收盘价低于阳线最低价 → 转化为看跌吞噬（更强）
- 如果只是略微回到阳线1/2以上 → 信号弱

## 3.4 刺透形态（Piercing Line）

**形态特征**：
1. 出现在下跌趋势底部
2. 第一根K线：长阴线
3. 第二根K线：长阳线
   - 开盘价低于前一天最低价（跳空低开）
   - 收盘价深入第一根阴线实体的上半部分（至少1/2以上）

**含义**：
- 看涨反转信号
- 乌云盖顶的反向对称形态
- 阳线收盘深入阴线实体越深越可靠
- 特别注意：如果阳线收盘仅到阴线的1/3以下，不能算刺透形态

## 3.5 看涨孕线（Bullish Harami）

**形态特征**：
1. 出现在下跌趋势底部
2. 第一根K线：长阴线（"母"线）
3. 第二根K线：小阳线（"子"线），实体完全被第一根阴线实体包裹

**含义**：
- 较弱的看涨反转信号
- 显示下跌动能减弱
- 需要后续阳线确认
- 第二根线实体越小，反转潜力越大（极端：十字孕线）

## 3.6 看跌孕线（Bearish Harami）

**形态特征**：
1. 出现在上涨趋势顶部
2. 第一根K线：长阳线
3. 第二根K线：小阴线，实体完全被第一根阳线包裹

**含义**：
- 较弱的看跌反转信号
- 显示上涨动能衰竭
- 需后续阴线确认

## 3.7 十字孕线（Harami Cross）

**形态特征**：
- 第一根K线：长实体线
- 第二根K线：十字星（实体极小），被第一根完全包裹

**含义**：
- 比普通孕线更强
- 十字星代表趋势的极端犹豫，预示变盘

## 3.8 双K线形态强弱排序

**看涨（由强到弱）**：
1. 看涨吞噬（最强）
2. 刺透形态
3. 十字孕线
4. 看涨孕线（最弱）

**看跌（由强到弱）**：
1. 看跌吞噬（最强）
2. 乌云盖顶
3. 十字孕线
4. 看跌孕线（最弱）
