# 第一章：K线基础与结构

## 1.1 K线的四大要素

每根K线包含四个关键价格信息：
- **开盘价（Open）**：该时间周期内的第一笔成交价格
- **收盘价（Close）**：该时间周期内的最后一笔成交价格
- **最高价（High）**：该时间周期内达到的最高价格
- **最低价（Low）**：该时间周期内达到的最低价格

### 阳线（Bullish Candle）
- 收盘价 > 开盘价
- 通常标记为绿色/白色/空心
- 矩形实体底部 = 开盘价，顶部 = 收盘价

### 阴线（Bearish Candle）
- 收盘价 < 开盘价
- 通常标记为红色/黑色/实心
- 矩形实体顶部 = 开盘价，底部 = 收盘价

## 1.3 实体的意义

实体长度代表多空双方的力量对比：
- **长实体**：一方主导市场，趋势强劲
- **短实体**：多空力量接近平衡，市场犹豫
- **无实体（十字星）**：开盘收盘完全一致，极致的平衡

量化定义（量化交易视角）：
- `实体 = |收盘价 - 开盘价|`
- 大实体：实体长度 > 近期ATR（平均真实波幅）的1.5倍
- 小实体：实体长度 < 近期ATR的0.3倍

## 1.4 影线的意义

上影线 = 最高价 - max(开盘价, 收盘价)
下影线 = min(开盘价, 收盘价) - 最低价

影线的核心含义：
- **长上影线** → 价格被推高后遭遇强烈卖压，"价格拒绝"高位
- **长下影线** → 价格被打压后遭遇强烈买盘，"价格拒绝"低位
- **短/无影线** → 单边市，一方完全没有抵抗
- **上下影线等长** → 多空激烈博弈，趋势不明

日本米商本间宗久（Munehisa Homma）在18世纪发明蜡烛图时，就通过影线判断市场心理："上影线如墓碑，下影线如地基"。
