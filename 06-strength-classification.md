# 第六章：K线形态强弱分级

## 6.1 单根K线强弱分级

### 看涨强弱（由强到弱）

**★★★★★（极强）**
1. **光头光脚大阳线**（Marubozu）：开盘=最低，收盘=最高，实体极长，显示极度强势
2. **锤子线 + 放巨量**：下跌底部出现长下影线配合放量

**★★★★（强）**
3. **大阳线**：实体长但有小影线
4. **蜻蜓十字星在底部**：下影线极长，拒绝下跌

**★★★（中等）**
5. **倒锤子线**：需次日阳线确认
6. **锤子线（阴线实体）**：阳线锤子 > 阴线锤子

**★★（弱）**
7. **小阳线纺锤线**：实体小，方向不明
8. **十字星**：中性信号，需结合位置

## 6.4 Bulkowski可靠性排名参考

根据Thomas Bulkowski对数千个形态的统计研究：

- 看涨吞噬：在牛市中可靠率约74%，熊市中约67%
- 启明星：总体可靠率约72%
- 锤子线：约68%（需确认）
- 刺透形态：约66%
- 孕线：约55%-60%（较弱）
- 十字星：约55%（高度依赖位置）

**重要结论**：
1. 任何形态都不是100%可靠的
2. 形态 + 位置 + 成交量 + 大周期趋势 = 高概率
3. 在牛市中看涨形态更可靠，熊市中看跌形态更可靠
4. 止损是必须的，形态只是概率优势
