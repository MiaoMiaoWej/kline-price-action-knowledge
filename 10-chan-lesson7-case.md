# 第十章：缠师第7课实战案例（年线突破+缩量回调+放量启动）

## 10.1 年线突破策略

**缠师原文（第7课·核心段落）**：

> "如果你的跟盘技术可以，就要在大盘回档的时候跟进强势股票。如果你的跟盘技术不行，有一种方式是最简单的，那就是盯着所有**放量突破上市首日最高价的新股**以及**放量突破年线然后缩量回调年线的老股**，这都是以后的黑马。特别那些年线走平后向上出现拐点的股票，一定要看好了。至于还在年线下面的股票，先别看了，等它们站上年线再说。其实这就是在牛市中最简单可靠的找所谓牛股的方法。"

### 策略三要素

1. **放量突破年线**：股价从年线下方首次有效突破250日均线，且突破时有明显放量
2. **缩量回调年线**：突破后股价回踩年线附近，成交量显著萎缩
3. **再次放量启动**：在年线附近获得支撑后，再次放量上涨，开启主升浪

## 10.2 经典案例：宝钢股份（600019）

**缠师原文案例**：

> "举一个例子，去看看宝钢，突破年线后缩量回调，10月23日回调4.20元，当时年线就在4.17元，然后再放量启动，今天，11月16日，已经6元多了，50%就这么完成了。从年线上启动，涨个50%，不像玩一样？"

### 案例详解

**时间**：2006年10月-11月

**第一步：放量突破年线**
- 宝钢股价从低位启动，放量突破250日均线（年线）
- 突破时成交量显著大于前期均量
- 这是"表态"：主力资金进场

**第二步：缩量回调年线**
- 突破后股价自然回落
- 2006年10月23日回调至4.20元
- 当时年线位置就在4.17元
- 回调到年线附近获得精确支撑
- 成交量极度萎缩（缩量）
- 这是"确认"：洗盘完成，浮动筹码出清

**第三步：放量启动**
- 在年线支撑位企稳后
- 再次放量拉升
- 11月16日已经涨至6元多
- 短短不到一个月涨幅超过50%
- 这是"进攻"：主升浪开启

## 10.3 策略的核心逻辑

### 为什么这个策略有效？

1. **年线的意义**：250日均线是牛熊分界线，突破年线意味着长期趋势从空头转向多头
2. **放量突破的意义**：主力资金真金白银买入，是"真心"而非"假突破"
3. **缩量回调的意义**：
   - 缩量 = 抛压衰竭，持股者惜售
   - 回踩年线不破 = 年线从阻力变成支撑
   - 这是"空中加油"，为下一波上涨积蓄力量
4. **再次放量的意义**：主力完成洗盘后发动主攻

### 量化指标参考
- 突破日成交量 > 20日均量的1.5倍
- 回调日成交量 < 20日均量的0.5倍（地量）
- 回调不破年线（允许小幅下影线刺破）
- 再次启动日成交量 > 20日均量

## 10.4 扩展应用：年线走平+向上拐点

缠师特别强调：**"特别那些年线走平后向上出现拐点的股票，一定要看好了"**

### 年线走平的含义
- 年线从下行转为走平 = 长期趋势从下跌转为震荡
- 年线向上拐头 = 长期趋势转为上升
- 此时年线附近买入，成本与主力相近

### 年线的三种状态
| 年线状态 | 含义 | 操作建议 |
|---------|------|---------|
| 年线下行 | 长期空头市场 | 不参与（防狼术） |
| 年线走平 | 多空转换中 | 关注突破机会 |
| 年线上行 | 长期多头市场 | 积极寻找买点 |

## 10.5 防狼术

**缠师原文（第7课）**：

> "至于还在年线下面的股票，先别看了，等它们站上年线再说。"

**第7课·防狼术三原则**：
1. 所有MACD黄白线在0轴以下徘徊的不玩
2. 年线以下的股票不玩
3. 小盘股要注意（大盘股骗线少，小盘股骗线多）

### 防狼术的操作意义
- 年线以下 = 长期趋势仍为空头
- 年线以下出现的任何看涨K线形态，可靠性都会打折扣
- 因为上方层层套牢盘，上涨阻力极大
- 等站上年线后再操作，是"趋势确认"思维

## 10.6 策略的实战流程

```
┌─────────────────────────────────────────┐
│        年线突破选股策略流程              │
├─────────────────────────────────────────┤
│  1. 筛选年线走平或开始上行的股票          │
│  2. 等待股价放量突破年线                  │
│     ├─ 放量：成交量 > 20日均量 × 1.5      │
│     └─ 确认：连续3日收盘站上年线          │
│  3. 等待缩量回调至年线附近                │
│     ├─ 缩量：成交量 < 20日均量 × 0.5      │
│     └─ 支撑：不破年线（或略微假跌破）     │
│  4. 再次放量启动时买入                    │
│     └─ 放量：成交量 > 20日均量            │
│  5. 持仓直到出现顶分型或跌破5日线         │
│  6. 按节奏轮动换股                       │
└─────────────────────────────────────────┘
```

## 10.7 与K线形态的结合

在"缩量回调年线"时，如果出现以下K线形态，买入信号更强烈：

1. **年线处出现锤子线**：下影线触及年线后收回，锤子线实体位于年线上方
2. **年线处出现看涨吞噬**：年线附近出现经典的看涨吞噬组合
3. **年线处出现启明星**：三根K线在年线处构筑底部
4. **年线处出现底分型**（缠论）：第三根阳线站上5日线，确认底分型成立

**综合应用示例**：
> 当一只股票同时满足：
> 1. 年线走平上翘
> 2. 缩量回踩年线
> 3. 在年线处出现看涨吞噬（或锤子线、启明星）
> 4. MACD在0轴上方
> → 这是极高概率的买入机会
